## Usage
### Clone Repository & Directory 
```bash
git clone https://github.com/zjqingzun/scyl-Ganc.git
cd scyl-Ganc
```

### Check Environment & Package (Go, Ignite)
```bash
# go
go version

# ignite
ignite version
```
If you haven't installed Ignite CLI yet, please refer to the official [Ignite CLI installation guide](./cli/readme.md).

### Run Chain
```bash
cd sw/ob
ignite chain serve -r
```

#### Run Node (Ob Node)
***Mandatory:*** 
*The prerequisite is that the chain must be running first.*
```bash
# Open a new terminal
cd sw/ob
obd tx dex --help
```
